# Etch-a-sketch

### Technologies Used

- HTML5
- CSS
- JavaScript

### Outcome
- Simple toy where you can draw on board up to 64x64 grid.
- Rainbow mode, color picker, eraser.
- Oldschool design  inspired by the original toy.
